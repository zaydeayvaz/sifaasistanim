<h2>1. Toplanan Veriler</h2>
<p>Şifa Asistanım uygulaması aşağıdaki verileri toplar:</p>
<ul>
  <li>İlaç adı, dozu, saati ve tedavi planı</li>
  <li>İlaç alındı/alınmadı geçmişi</li>
  <li>E-posta adresi (hesap oluşturulursa)</li>
  <li>Cihazda girilen sağlık ölçüm notları (yalnızca yerel)</li>
</ul>

<h2>2. Verilerin Kullanımı</h2>
<p>Toplanan veriler yalnızca şu amaçlarla kullanılır:</p>
<ul>
  <li>İlaç hatırlatma bildirimleri göndermek</li>
  <li>Kullanıcının onayladığı yakınıyla ilaç planını paylaşmak</li>
  <li>Uygulama işlevselliğini sağlamak</li>
</ul>

<h2>3. Veri Paylaşımı</h2>
<p>Verileriniz üçüncü taraflarla satılmaz veya pazarlama amacıyla paylaşılmaz. Yalnızca kullanıcının açık onayıyla belirlediği kişilerle ilaç planı paylaşılır.</p>

<h2>4. Sağlık Verileri</h2>
<p>Sağlık sekmesine girilen tüm ölçüm kayıtları (tansiyon, kan şekeri, kilo vb.) yalnızca cihazda tutulur. Bu veriler hiçbir sunucuya gönderilmez.</p>

<h2>5. Veri Güvenliği</h2>
<p>Bulut senkronizasyonu Supabase altyapısı üzerinden şifreli bağlantıyla yapılır. Verilerinize yalnızca hesabınızla erişilebilir.</p>
